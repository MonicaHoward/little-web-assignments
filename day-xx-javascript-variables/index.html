JAVASCRIPT VARIABLES

BASIC VARIABLE SYNTAX

var

This is the variable declaration which indicates that you are initializing variable.

city

This is the variable name. One word variable names should be in all lowercase letters.Once the variable has been declared with the "var" keyword you should reference this variable by its variable name only (without using the "var" keyword).

=

This is the equals sign operand. It assigns a value to the variable.

Roanoke;

This is the value of the variable. This value will be stored inside of the variable. Don't forget the semicolon!

CAMEL CASE

var

cityPopulation

=

6,974;

If your variable is more than one word each subsquent word in the variable name should be started with an uppercase letter.

EXAMPLE WITH A NUMBER

var

states

=

50;

When assigning numerical values to a variable be sure to represent numerical values as the actual number.
